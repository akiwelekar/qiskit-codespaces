# Qiskit Learning Portfolio

> A structured journey from quantum computing fundamentals to quantum software architecture using Qiskit.

---

## About this Repository

This repository documents my journey of learning, experimenting with, and applying **Qiskit** to quantum computing.

The repository serves four purposes:

- Learn Qiskit systematically through executable examples.
- Develop practical skills in quantum programming.
- Explore quantum algorithms and hybrid quantum-classical computing.
- Investigate software engineering principles for quantum software systems through the **Quantum Software Architecture Framework (QSAF)**.

The repository is continuously updated as part of my research and professional development.

---

# Repository Objectives

- Learn Qiskit from first principles
